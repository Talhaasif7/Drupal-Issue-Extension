# IssueSniper: High-Fidelity Drupal Issue Tracker 🏹⚡

Never miss a claimable Drupal issue again. Get instant Chrome alerts the second a new issue is claimable.

## 🚀 Key Features

- **Professional Web Landing Page**: A stunning entry point for your project.
- **Cyber-Minimalist Extension**: A dedicated "Command Center" for monitoring project activity.
- **Auto-Fetch Discovery**: Automatically tracks active Drupal projects if your watchlist is empty.
- **Instant Notifications**: Real-time native OS alerts for claimable issues.

## 🛠️ Tech Stack

- **Frontend**: React + Vite + Vanilla CSS (Glassmorphism)
- **Backend**: Supabase (PostgreSQL + Real-time)
- **API**: Drupal.org REST API

## 📦 Getting Started

### 1. Prerequisites
- Node.js (v18+)
- A Supabase Project (URL and Anon Key)

### 2. Setup
1. Clone the repository.
2. Run `npm install` to restore dependencies.
3. Copy `.env.example` to `.env` and add your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=your-project-url
   VITE_SUPABASE_ANON_KEY=your-anon-key
   ```
4. Run the Supabase schema in `supabase_schema.sql` in your Supabase SQL Editor.

### 3. Development
Run the local dev server (includes Landing Page and Extension previews):
```bash
npm run dev
```
Preview at: `http://localhost:5173/`

### 4. Production Build
Build both the Landing Page and Extension assets:
```bash
npm run build
```

### 5. Install Extension
1. Open Chrome and go to `chrome://extensions`.
2. Enable "Developer Mode".
3. Click "Load unpacked" and select the `dist` folder generated by the build.

## 📂 Project Structure
- `index.html`: The web landing page.
- `popup.html`: The extension's internal popup UI.
- `src/App.tsx`: The React logic for the extension.
- `src/background.ts`: The Chrome background worker for periodic polling.
- `src/lib/drupal-api.ts`: API client for Drupal.org.
